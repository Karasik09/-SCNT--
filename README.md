# Эпигенетические маркеры жизнеспособности клонированных эмбрионов

Анализ данных метилирования ДНК из датасета [GSE61279](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE61279) (Matoba et al., Cell Stem Cell, 2014).

## Что сделано
- Проведён анализ профилей метилирования у 8 мышиных эмбрионов (4 SCNT, 4 контрольных)
- Выявлены CpG-сайты, ассоциированные с жизнеспособностью эмбрионов
- Подтверждена связь с регионами генов *Igf2* и *H19*

## Как запустить
1. Открыть [`notebook.ipynb`](notebook.ipynb) в Google Colab  
2. Нажать **Runtime → Run all**
